 # Welcome to biosnicar!

`biosnicar` is free and open source software for simulating energy fluxes in snow and glacier ice.

`biosnicar` is available as [Python code](https://github.com/jmcook1186/biosnicar-py) that you can easily download and run yourself, or as a [web application](bit.ly/biosnicar) that can be run code-free in your browser.

You can visit the [About](about) pages to learn about the theoretical background to `biosnicar`.

To get up and running with `biosnicar`, follow the [Quick Start guide](quick-start)

You can navigate through the documentation using the menu in the left sidebar. 
See the `guides` section to learn about key components of `biosnicar`.
See `Modules` to see reference documentation for every class and function in the `biosnicar` source code.
