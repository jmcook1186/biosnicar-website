# Examples

Worked code examples covering the full BioSNICAR workflow. These examples correspond to the scripts in the [examples/](https://github.com/jmcook1186/biosnicar-py/tree/main/examples) directory of the repository. Each can be run from the repository root:

```sh
python examples/01_basic_forward_model.py
python examples/05_emulator_predict.py --plot
python examples/10_end_to_end_workflow.py --plot
```

## Overview

| # | Topic | What you'll learn |
|---|-------|-------------------|
| 01 | [Forward Model](/examples/forward-model) | `run_model()` with defaults, overrides, multi-layer, spectral output |
| 02 | [Parameter Sweeps](/examples/forward-model#parameter-sweeps) | 2D/3D sweeps, impurity sweeps, spectral output |
| 03 | [Remote Sensing](/examples/remote-sensing-examples) | `.to_platform()` chaining, GCM bands, multi-platform |
| 04 | [Building Emulators](/examples/emulator-examples) | Custom emulator training, accuracy validation |
| 05 | [Emulator Prediction](/examples/emulator-examples#predicting-with-the-emulator) | Load, predict, batch prediction, speed comparison |
| 07 | [Spectral Inversion](/examples/inversion-examples) | SSA retrieval from field spectrometer data |
| 08 | [Satellite Inversion](/examples/inversion-examples#satellite-band-retrieval) | Band-mode retrieval from Sentinel-2, Landsat, MODIS |
| 10 | [End-to-End](/examples/end-to-end) | Full pipeline from observation to physical interpretation |
| 11 | [Subsurface Light](/examples/forward-model#subsurface-light) | PAR depth profiles, spectral heating rates |

## Dependencies

- Examples 01–03 need only the base BioSNICAR dependencies
- Examples 04 and 06 require `scikit-learn` (for `Emulator.build()`)
- Example 09 with `--mcmc` requires `emcee`
- `--plot` requires `matplotlib`
- Examples 05–10 require the pre-built default emulator at `data/emulators/glacier_ice_8_param_default.npz`
